Classyfing price categories for Airbnb

Classify listings from Airbnb into price categories using the listings text and image data simultaneously. (Key concepts: multi-objective, multi-modality, NLP, computer vision, RNN, CNN, TensorFlow).

Problem Formulation

One challenge individuals may encounter when creating a new Airbnb listing is determining the appropriate asking price. In this project, I won't be predicting the precise listing price based on property features, as it is a relatively straightforward task. Instead, the objective is to categorize each listing into three tiers: beginner, plus, and premium, providing guidance on pricing. To achieve this, I will employ a multi-objective (multi-task) multi-modality approach, utilizing both text and image data. Natural language processing (NLP) will be applied to the listing summary (text data), and convolutional neural network (CNN) models will be constructed to extract features from listings images.

The dataset for this project contains listing of various regions in Montreal. I will experiment with various recurrent neural network (RNN) architectures, including GRUs (gated recurrent units) and LSTM (long short-term memory) models, to handle the text data. Simultaneously, different convolutional neural network architectures, such as VGG and ResNet, will be employed to process the image data. Both networks would then be combined to generate a single prediction.
